# Flask-MarketPlace

A marketplace web application built with Flask as part of my journey
into backend development with Python.

## Features
- User authentication (register/login)
- Product listing and marketplace browsing
- Basic marketplace logic

## Tech Stack
- Python
- Flask
- Jinja2
- Designed database models using SQLAlchemy ORM
- HTML/CSS
